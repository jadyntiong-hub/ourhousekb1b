Daily Activities

One small job each day · 每天一个小任务

Monday

Vocabulary cards + room word game + Recording 1

Goal: Children can say and recognise the six room words.
目标：认识并读出 6 个房间词。

Tuesday

Missing letters + memory spelling + examiner dictation

Goal: Children can hear slow lowercase letter names and write from memory.
目标：听小写字母名称并凭记忆写词。

Wednesday

Unscramble words + grammar choice

Goal: Children can use What is he/she doing? He is / She is ...
目标：正确使用他/她正在做什么的句型。

Thursday

/m/ phonics + singular/plural check + recording

Goal: Children can hear /m/ clearly and practise a room / rooms.
目标：清楚听读 m，并练习单数/复数。

Friday

Exam Master + Tracker screenshot + take-home mini test

Goal: Children can apply listening, reading, writing and speaking practice.
